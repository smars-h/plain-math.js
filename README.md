# Simple Math! # 

# Installation #
you can install it by typing `npm i plain-math.js`

# Guide #
You can use it like this

```js
const calc = include('plain-math.js');

let result = calc.add(2, 3);

console.log(result);
``` 

# All methods #
Add
```js
calc.add(x, y);
```

Subtract
```js
calc.sub(x, y); 
```

Divade
```js
calc.div(x, y);
```

Multiply
```js
calc.multi(x, y) 
```

Cosinus
```js
calc.cos(x); 
```

Tanget 
```js
calc.tan(x); 
```

Absolute
```js
calc.abs(x);
```
